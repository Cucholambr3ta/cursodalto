1. Estructura Básica de HTML5

Comando útil: Escribiendo html:5 en VS Code te crea automáticamente esta estructura básica:

Estructura completa HTML5:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

Explicación de cada parte:

  • <!DOCTYPE html>: Declara que es un documento HTML5
  • <html lang="en">: Elemento raíz, lang especifica el idioma
  • <head>: Contiene metadatos no visibles en la página
  • <body>: Contiene todo el contenido visible de la página

2. Meta Tags Importantes

charset="UTF-8"

Define la codificación de caracteres del documento. UTF-8 soporta caracteres especiales y acentos correctamente.

viewport

Configura la visualización en dispositivos móviles:

  • width=device-width: hace que el ancho de la página coincida con el ancho de la pantalla del dispositivo
  • initial-scale=1.0: establece el nivel de zoom inicial al 100% cuando la página se carga por primera vez

Esto es esencial para que la página se vea correctamente en teléfonos y tablets (diseño responsive).

3. Comentarios en HTML

Los comentarios en HTML se escriben así: <!-- Tu comentario aquí -->

Características de los comentarios:

  • No se muestran en el navegador
  • Son útiles para documentar el código
  • Ayudan a otros desarrolladores a entender tu código
  • Puedes usar múltiples líneas

Ejemplo de comentarios:

<!-- Este es un comentario de una línea -->

<!-- Este es un comentario
     de múltiples líneas
     que explica algo importante -->

4. Jerarquía de Encabezados

Comando útil: Si escribes h$* cantidad de elementos h necesarios te agrega automáticamente los elementos h1, h2, h3, etc. de manera ordenada.

Uso correcto de encabezados:

  • H1: Solo se utiliza 1 en la página web, será el encabezado más importante, lo que describe de qué tratará la página
  • H2: Se utiliza para secciones o subtítulos
  • H3: Se utiliza para subsecciones
  • H4, H5, H6: Se utilizan para subsecciones más específicas

Esto se utiliza en orden jerárquico

6. Etiquetas de Formato de Texto

HTML ofrece varias etiquetas para dar formato y significado al texto:

Etiquetas de énfasis y importancia:

  • <strong>: Texto importante o con énfasis fuerte
  • <em>: Texto enfatizado (cursiva)
  • <b>: Texto en negrita (solo visual)
  • <i>: Texto en cursiva (solo visual)

Etiquetas para código y texto técnico:

  • <code>: Fragmentos de código en línea
  • <pre>: Texto preformateado (mantiene espacios y saltos)
  • <kbd>: Teclas del teclado
  • <var>: Variables

Otras etiquetas útiles:

  • <mark>: Texto resaltado
  • <small>: Texto pequeño
  • <sub>: Subíndice
  • <sup>: Superíndice
  • <del>: Texto tachado
  • <ins>: Texto insertado (subrayado)

7. Enlaces

Los enlaces se crean utilizando la etiqueta <a>. El atributo href especifica la URL a la que se dirige el enlace.

Ejemplo de código y resultado:

Código: <a href="https://www.ejemplo.com">Ejemplo</a>

Resultado: Ejemplo

Importante: El enlace siempre debe comenzar con https:// para que sea dirigido a una página externa, de lo contrario se asumirá que es un enlace interno dentro de alguna carpeta del proyecto.

8. Tablas en HTML

Las tablas se utilizan para mostrar datos organizados en filas y columnas. Son muy útiles para comparaciones y mostrar información estructurada.

Estructura básica de una tabla:

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>

Elementos de una tabla:

  • <table>: Contenedor principal de la tabla
  • <tr>: Table Row - Fila de la tabla
  • <th>: Table Header - Encabezado de columna (se ve en negrita)
  • <td>: Table Data - Celda de datos

Tabla básica:

Etiqueta Descripción
<h1> Encabezado principal
<p> Párrafo

Tabla con estilos mejorados:

Para que la tabla se vea mejor, puedes agregar atributos de estilo:

<table border="1" style="border-collapse: collapse; width: 100%;">

Atributos útiles:

  • border="1": Agrega bordes a la tabla
  • style="border-collapse: collapse": Une los bordes para que se vean mejor
  • style="width: 100%": Hace que la tabla ocupe todo el ancho disponible

9. Entidades HTML - Mostrando Código

Para mostrar código HTML en una página web sin que se ejecute, necesitamos usar entidades HTML:

Entidades principales:

  • &lt; - Representa el símbolo < (less than / menor que)
  • &gt; - Representa el símbolo > (greater than / mayor que)
  • &amp; - Representa el símbolo & (ampersand)
  • &quot; - Representa las comillas "

¿Por qué son necesarias?

El navegador interpreta < y > como inicio y fin de etiquetas HTML. Si queremos mostrar estas etiquetas como texto, debemos "escaparlas":

Ejemplos comparativos en tabla:

Lo que escribes en el código Lo que se ve en el navegador
&lt;h1&gt;Título&lt;/h1&gt; <h1>Título</h1>
&lt;strong&gt;Texto&lt;/strong&gt; <strong>Texto</strong>
&lt;a href="enlace.html"&gt;Link&lt;/a&gt; <a href="enlace.html">Link</a>
