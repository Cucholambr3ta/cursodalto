Enlaces

Los enlaces se crean utilizando la etiqueta <a>. El atributo href especifica la URL a la que se dirige el enlace. El enlace siempre debe comenzar con https://, http:// o también con // para que sea dirigido a una página web externa; de lo contrario, se asumirá que es un enlace a un archivo dentro de alguna carpeta local. Por buenas prácticas, es recomendable que los enlaces externos comiencen con https:// para garantizar una conexión segura. También puedes escribir solo el comando a y así puedes crear una estructura de enlace sin necesidad de escribir el comando <a href=""></a>. para abrir el enlace en una nueva pestaña, puedes agregar el atributo target="_blank" al enlace.

Ejemplo de código: <a href="https://www.ejemplo.com">Ejemplo</a>
